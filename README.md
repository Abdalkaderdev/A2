### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdalkaderdev/A2.git
   cd A2
   ```

2. Open `index.html` in a web browser or serve locally:
   ```bash
   # Using Python

   npx serve .
   ```

- **Styling**: Customize appearance by editing `style.css`
